Live Weather Viewer
Overview
Welcome to the Live Weather Viewer project!
This application allows users to instantly check live weather conditions by simply entering the city name in the input field.
The user-friendly design makes it easy for anyone to access real-time weather information, including humidity, temperature, and wind conditions.

Features
City-based Weather Updates:

Users can input the city name of their choice.
Receive live updates on humidity, temperature, and wind conditions for the specified city.
User-Friendly Design:

Intuitive and straightforward interface for easy navigation.
Minimalistic design ensures a seamless and enjoyable user experience.

Stay updated with the latest weather effortlessly! 🌦️
